MNF – Backend

Este repositório contém todo o backend do projeto MNF, incluindo a estrutura completa da API, regras de negócio e integração com o banco de dados.

Tecnologias utilizadas

Java

Spring Boot

Spring Data JPA

MySQL

Maven

Git & GitHub

Estrutura do projeto

Controllers – Camada responsável por receber e responder às requisições.

Services – Regras de negócio e lógica da aplicação.

Repositories – Comunicação direta com o banco de dados.

Entities – Mapeamento das tabelas e modelos do sistema.

Funcionalidades principais

CRUD completo

Gestão de usuários

Operações específicas do projeto MNF

Como executar

Clone este repositório

Configure o banco no application.properties

Execute o projeto com:

mvn spring-boot:run
